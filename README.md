# Sales Performance Analysis

This project analyzes sales performance using the Sample Superstore dataset.

## Project Structure

- `data/raw/` stores the original dataset.
- `sql/` contains table creation and analysis queries.
- `screenshots/` stores exported visuals and KPI screenshots.

## Workflow

1. Place `Sample - Superstore.csv` in `data/raw/`.
2. Run the SQL scripts in order from the `sql/` folder.
3. Add final dashboard or KPI screenshots to `screenshots/`.
